Highlighter

To incorporate this script into your project simply wrap the content which you want to be shareable in a HTML "DIV" tag with an "ID" value of "CONTENT".

Highlight the text with your mouse to display Social Media share buttons.

Share the selected text on Twitter or Facebook.

Use repeatedly on the same page. Just assign the ID "CONTENT" to any content.

Style the buttons however you like. Icons are FontAwesome glyphs and basic CSS styling.

Share highlighted text across blocks.
